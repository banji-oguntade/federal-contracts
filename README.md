# Federal IT Contracting Roadmap

Interactive checklist app for planning a federal IT contracting strategy. Built with React + Vite. Progress is saved automatically in the browser (localStorage).

## Run locally

Requires Node.js 18+.

```bash
npm install
npm run dev
```

Open the URL Vite prints (usually http://localhost:5173).

## Build for production

```bash
npm run build
```

Output goes to `dist/` — deployable to any static host (Netlify, Vercel, GitHub Pages, S3).
